<objective>
Close the schema gap that causes all facebook publishes to be incorrectly blocked by the Phase 11 music license gate.

Purpose: Phase 11 code already references platform_facebook (platform_publish.py lines 102, 115, 117), but migration 0008 only created 3 platform columns. Without platform_facebook in the schema, track.get('platform_facebook', False) always returns False, blocking every facebook publish regardless of actual license status. This plan adds the missing column, syncs fixture data to match 4-platform reality, and adds a targeted test to prove the gate works correctly for facebook.

    def test_license_gate_facebook_blocked():
        """PUB-01 gap closure: track with platform_facebook=False is blocked for facebook."""
