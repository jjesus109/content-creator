<objective>
Implement the approval timeout flow: VideoStatus enum extension, 24-hour last-chance job, pipeline-start stale-row cleanup, and timeout job cancellation on approval.

Purpose: Prevents the pipeline from hanging indefinitely on unanswered approvals. After 24h without a response, the creator gets a last-chance Telegram message; at the next pipeline run any still-unanswered row is marked approval_timeout and new generation proceeds.
Output: approval_timeout.py job file, VideoStatus extended, daily_pipeline_job startup check, send_approval_message_sync schedules the timeout, handle_approve cancels it.
</objective>
