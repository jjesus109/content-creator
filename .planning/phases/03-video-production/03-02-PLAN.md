**`HeyGenService` class:**
- `__init__(self)` — no arguments; reads settings via `get_settings()` internally
- `submit(self, script_text: str, background_url: str) -> str` — POSTs to `https://api.heygen.com/v2/video/generate`, returns the `video_id` string

```python
payload = {
    "video_inputs": [{
        "character": {
            "type": "avatar",

    "callback_url": settings.heygen_webhook_url,
```

- If pool has only one URL, return it (cannot enforce no-repeat with single option)
- Filters out `last_used_url` from pool; if filtered pool is empty (should not happen), fall back to full pool
- Returns `random.choice(available)`

Create `src/app/services/video_storage.py` with a single `VideoStorageService` class.

**`VideoStorageService` class:**
- `__init__(self, supabase=None)` — accepts optional supabase client for testability (follows SimilarityService pattern from Phase 2); if None, calls `get_supabase()` internally
- `upload(self, video_bytes: bytes, target_date: date | None = None) -> str` — uploads to Supabase Storage, returns stable public URL

**`upload()` implementation:**
```python
from datetime import date
from app.services.database import get_supabase

VIDEO_BUCKET = "videos"

def upload(self, video_bytes: bytes, target_date: date | None = None) -> str:
    if target_date is None:
        target_date = date.today()
    file_path = f"videos/{target_date.isoformat()}.mp4"

    self.supabase.storage.from_(VIDEO_BUCKET).upload(
        path=file_path,
        file=video_bytes,
        file_options={
            "content-type": "video/mp4",
            "upsert": "true",          # overwrite on same-day re-run (upsert decision: Claude's discretion)
            "cache-control": "31536000",  # 1 year — content is permanent once approved
        },
    )
    public_url = self.supabase.storage.from_(VIDEO_BUCKET).get_public_url(file_path)
    logger.info("Uploaded video to Supabase Storage: %s", public_url)
    return public_url
```

Key constraints:
- `upsert: "true"` (string, not bool) — Supabase Python client expects string per official docs
- File path `videos/YYYY-MM-DD.mp4` is the locked naming convention
- NEVER pass the HeyGen signed URL to this method — callers pass `bytes`, method returns the stable URL
- The `VIDEO_BUCKET = "videos"` bucket must be PUBLIC in Supabase dashboard (pre-flight note in docstring)

Add module-level logger: `logger = logging.getLogger(__name__)`. Import `logging`, `from datetime import date`, `from app.services.database import get_supabase`.
