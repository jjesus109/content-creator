Create `migrations/0004_approval_events.sql` with:

**Section A — approval_events table:**
```sql
CREATE TABLE IF NOT EXISTS approval_events (
    id                  uuid PRIMARY KEY DEFAULT gen_random_uuid(),
    created_at          timestamptz DEFAULT now() NOT NULL,
    content_history_id  uuid NOT NULL REFERENCES content_history(id),
    action              text NOT NULL CHECK (action IN ('approved', 'rejected')),
    cause_code          text,
    CONSTRAINT rejection_requires_cause CHECK (
        action = 'approved' OR cause_code IS NOT NULL
    )
);
```

cause_code is NULL for approvals and one of: 'script_error', 'visual_error', 'technical_error', 'off_topic' for rejections.

**Section B — indexes:**
```sql
CREATE INDEX IF NOT EXISTS approval_events_content_history_id_idx
    ON approval_events(content_history_id);

CREATE INDEX IF NOT EXISTS approval_events_created_at_idx
    ON approval_events(created_at);
```

approval_events_content_history_id_idx supports `is_already_actioned()` lookup.
approval_events_created_at_idx supports `get_today_rejection_count()` date-range query.

**Section C — post_copy column on content_history:**
