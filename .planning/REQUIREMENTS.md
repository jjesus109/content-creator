# Requirements: Autonomous Content Machine

**Defined:** 2026-03-18
**Milestone:** v2.0 — Mexican Cat Content Machine
**Core Value:** A cute Mexican cat video lands in Telegram every day, ready to approve and publish — the creator's only job is to say yes or no.

## v2.0 Requirements

Requirements for the Mexican Cat Content Machine milestone. Each maps to roadmap phases.

### Video Generation

- [ ] **VID-01**: System generates cat video using Kling AI 3.0 via fal.ai (20-30s, 9:16 1080p) replacing HeyGen
- [ ] **VID-02**: Fixed Mexican cat character defined via Character Bible (40-50 word trait spec) embedded in every generation prompt
- [ ] **VID-03**: Kling API circuit breaker: pauses pipeline if >20% failure rate; exponential backoff (2s, 8s, 32s); credit balance checked before each call
- [ ] **VID-04**: AI content label applied on all platforms before video is published (mandatory TikTok/YouTube/Instagram compliance)

### Scene Engine

- [ ] **SCN-01**: Daily scene prompt generated by GPT-4o selecting from curated library of 40-60 location + activity + mood combinations
- [ ] **SCN-02**: Seasonal calendar injects themed scene overlays for Sep 16 (Independence Day), Nov 1-2 (Day of Dead), Nov 20 (Revolution Day), Aug 8 (Cat Day)
- [ ] **SCN-03**: Anti-repetition check via pgvector blocks scene >75-80% similar to any video in past 7 days (recalibrated from v1.0's 85%)
- [ ] **SCN-04**: Rejected scene details stored and injected into next generation prompt as negative context
- [ ] **SCN-05**: Scene prompt produces universal Spanish caption (5-8 words; [observation] + [implied personality] formula)

### Music

- [ ] **MUS-01**: Pre-curated music pool of 200+ tracks stored in DB, tagged by mood, tempo, and per-platform license status
- [ ] **MUS-02**: Music track selected per video by matching scene mood to BPM range (playful: 110-125, sleepy: 70-80, curious: 90-100)
- [ ] **MUS-03**: Per-platform license matrix maintained; tracks validated as cleared for TikTok, YouTube, and Instagram independently

### Publishing

- [ ] **PUB-01**: Music license validated against matrix before publishing; publish blocked if track not cleared for target platform

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Multiple cat characters | Brand loyalty drops 40% when character switches; single character locked |
| Voiceover / TTS | Defeats universal appeal; visual-only is cat content's strength |
| Spoken philosophical content | v2.0 is visual-only — cat does not speak |
| Weekly mood profile Telegram flow | Replaced by seasonal calendar; not applicable to cat content |
| 5 Pillars philosophical framework | Not applicable to cat content |
| Per-platform post copy variants | Universal caption first; no data supports ROI of variants |
| Analytics dashboards / UI | All interaction via Telegram only |
| Multi-user support | Single creator only |
